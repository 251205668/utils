# 封装的一些常用工具库

### Installation

Using npm:

```bash
$ npm i typedash
```

### Usage

From TypeScript/es6 (using webpack2 or rollup)

```typescript
import { noop } from "typedash";

noop();
```

From nodejs

```js
const { noop } = require("typedash");

noop();
```

[Full documentation](https://arusanov.github.io/typedash/)
